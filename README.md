# Next.js Task Manager

Application simple de gestion de tâches avec Next.js App Router, TypeScript, Tailwind CSS et localStorage.

## Fonctionnalités

- Ajouter une tâche
- Empêcher les tâches vides
- Marquer comme terminée / en cours
- Supprimer une tâche
- Filtrer par Toutes, Terminées, En cours
- Persistance avec localStorage
- UI responsive avec dark mode

## Lancer le projet

```bash
npm install
npm run dev
```

Puis ouvrir :

```bash
http://localhost:3000
```
